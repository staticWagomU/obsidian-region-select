# Region Select

[English](./README.md)

Obsidian用のEmacs風マーク＆ポイントによるテキスト選択プラグイン。特にモバイルデバイス向けに設計されています。

## なぜこのプラグイン？

モバイルデバイスでのテキスト選択は本当に大変です。長押しジェスチャーは正確性に欠け、長い文章の選択ハンドルをドラッグするのはストレスがたまります。このプラグインはEmacs風のアプローチでこの問題を解決します：

1. **マークを設定** - 開始位置を記録
2. **自由にカーソル移動** - Obsidianのキーボードやナビゲーションを使用
3. **マークまで選択** - 間のテキストをすべて選択

タッチ選択との格闘はもう終わりです！

## 機能

| コマンド | アイコン | 説明 |
|---------|------|-------------|
| **Set Mark** | 📍 | カーソル位置を選択開始点として保存 |
| **Select to Mark** | 📝 | マークから現在のカーソルまでのテキストを選択 |
| **Clear Mark** | ✕ | 保存したマークを削除 |
| **Toggle Mark** | 🎯 | ワンタップ操作：マーク設定 → 移動 → 再タップで選択 |

### Toggle Mark（モバイルにおすすめ）

**Toggle Mark** コマンドは最もシンプルなワークフローを提供します：

1. 1回タップ → カーソル位置にマークを設定
2. カーソルを目的地まで移動
3. 再度タップ → テキストを選択してマークをクリア

リボンアイコンで現在の状態がわかります：
- `locate` アイコン：マーク未設定（設定待ち）
- `target` アイコン：マーク設定済み（選択待ち）

## 使い方

### 基本的なワークフロー

```
1. 選択開始位置にカーソルを配置
2. "Set Mark" コマンドを実行
3. 選択終了位置にカーソルを移動
4. "Select to Mark" コマンドを実行
5. テキストが選択されます！
```

### モバイル設定（推奨）

#### 方法1：モバイルツールバー
1. 設定 → モバイル → ツールバーボタンを管理
2. "Toggle mark" コマンドを追加
3. ツールバーからワンタップで選択！

#### 方法2：コマンドパレット
- 下にスワイプしてパレットを開く
- "mark" で検索してすべてのコマンドを表示

## インストール

### Obsidianコミュニティプラグインから

1. 設定 → コミュニティプラグイン を開く
2. 閲覧して "Region Select" を検索
3. インストールして有効化

### 手動インストール

1. [Releases](https://github.com/staticWagomU/obsidian-region-select/releases) から `main.js`、`manifest.json` をダウンロード
2. フォルダを作成：`VaultFolder/.obsidian/plugins/region-select/`
3. ファイルをフォルダにコピー
4. 設定 → コミュニティプラグイン で有効化

## 開発

```bash
# 依存関係のインストール
pnpm install

# 開発ビルド（ウォッチモード）
pnpm run dev

# 本番ビルド
pnpm run build

# テスト実行
pnpm test

# リント
pnpm run lint
```
